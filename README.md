# CREATION PROJET

À ce stade, le choix se fait entre "previous version" pour les projets qui bénéficient régulièrement de nouvelles versions, et "number of days" pour les projets organisés en sprints, où la livraison doit être effectuée à intervalles réguliers de x jours.
Dans notre situation, nous optons pour la première option et créons le projet, ce qui est amplement suffisant. Dans ce scénario, SonarCloud effectue une première analyse et vous fournit des retours.
